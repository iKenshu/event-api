# Event API

This is a simple API to manage events, create, read, update, and delete events. You can also register and login with certain endpoints and you can register to events.

## Usage

To use this API, you can use the following endpoints:

- `/docs`: To get the documentation of the API.
- `/graphql`: To get the GraphQL API. It's just a basic stuff for events

## Installation

To install this API, you can use docker compose:

```bash
docker compose build
docker compose up
```
